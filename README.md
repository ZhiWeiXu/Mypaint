###具体功能    

    ·美观大方的操作界面，保障系统的易用性。  
    ·详细的商品分类管理，易于顾客获取想要的信息。   
    ·客户登陆和注册。  
    ·订单管理。  
    ·用户管理。  
    ·商品搜索和购物车。  
    ·热卖商品展示。  
      
        
        
###编译需求  

    ·JDK1.7。  
    ·Android Studio 2.0  
    ·Bmob后端云服务平台
